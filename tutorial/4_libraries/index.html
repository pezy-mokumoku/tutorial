Tools & LibrariesÂ¶

You need permission to see following libraries. Please request your contact.

  • pzcomplex
    • A library for .pzc compatible with <complex> in C++ standard library.
    • We intend to merge this library into the next version of PZSDK.
  • pzsparse
    • Sparse matrix-Vector Multiplication Library.
    • CRS / CCS / COO format is supported.
  • pzBLAS (Under construction)
    • The basic linear algebra subroutines (BLAS) for PEZY-SCx series.
    • Including incomplete functions because it aims at PEZY-SC3.
  • pzfft (Under construction)
    • Fast Fourier Transform on PEZY-SCx series.
    • Including incomplete functions because it aims at PEZY-SC3.
