Introduction¶

Welcome! This document is PEZY-SCx series programming tutorial!

This document explains PEZY-SCx series specifications and programming model using the explanation of sample code.

PZCL is based on OpenCL 1.2 (OpenCL 1.2 Reference pages)

PEZY-SCx series has the following features:

  • Many-core processor (1024 / 2048 PEs) [Spec Sheet]
  • World’s best class energy efficiency (1st place of Green 500, Nov. 2018)
  • Fast scratchpad memory on PE (local memory) [Detail]
  • 8-way hardware multi-threading (Fine-Grain Multi-threading) [Detail]
  • MIMD (Multiple Instruction Stream Multiple Data Stream)

PEZY-SCx series is MIMD (multiple instruction, multiple data)-type processor. Each PE (Processing Element) has eight registers and eight program counters for running eight threads independently.

PEZY-SCx series processor is a three-layer hierarchical structure called “Prefecture,”’ “City,” and “Village” (they are PE groups). Each Village has 4 PEs, each City has 4 Villages, and each Prefecture has 16 Cities. PEZY-SC / SC2 has 4 / 8 Prefectures.

Features of PEZY-SC¶

Feature of PEZY-SC (2014~) is:

  • 28 nm process
  • 1024 PE, 733 MHz
  • 16 MB (16KB/PE) scratchpad memory
  • 1.5 TFLOPS (DP) / 3.0 TFLOPS (SP, dual issue)

For more details, refer to the following page. [Detail]

Features of PEZY-SC2¶

Feature of PEZY-SC2 (2017~) is:

  • 16 nm process
  • 2048 PE, 1GHz
  • 40 MB (20KB/PE) scratchpad memory
  • 64 bit SIMD
  • 4.1 TFLOPS (DP) / 8.2 TFLOPS (SP, x2 SIMD) / 16.4 (HP, x4 SIMD)

For mode details, refer to the following page. [Detail]
