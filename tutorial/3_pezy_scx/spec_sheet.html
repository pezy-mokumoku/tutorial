PEZY-SCx series Specification¶

Table1 shows specification of PEZY-SCx series processor.

Table 1 PEZY-SCx series Specification¶
  PEZY-SC PEZY-SC2
Process 28 nm 16 nm
The number of PEs 1024 2048
Clock Freq. 733 MHz 1 GHz
L1 Cache Memory 1 MB 4 MB (D), 8 MB (I)
L2 Cache Memory 4 MB 8 MB (D), 4 MB (I)
Last Level Cache Memory (LLC) 8 MB 40 MB
Scratchpad Memory (local memory) 16 MB (16KB/PE) 40 MB (20KB/PE)
PCIe Interface Gen3 8 Lane 4 Port Gen4 8 Lane 4 Port
DDR Interface DDR4 64 bit 8 ch. DDR4 64 bit 4 ch.
SIMD vector length N/A (scalar) 64 bit
Peak (DP, TFLOPS) 1.5 4.1
Peak (SP, TFLOPS) 3.0 (dual issue) 8.2 (x2 SIMD)
Peak (HP, TFLOPS) N/A 16.4 (x4 SIMD)
Power Consumption (Peak) 70 W 200 W

(*) Clock Freq., global memory size, # of PE, and etc. depend on the system configuration.
