Q & A¶

Kernel timeout error¶

If the following error occurs in pzcl.log on PEZY-SCx series:

<ERROR> 0 : WaitKernel failed:-1000
<ERROR> 0 : Kernel Event Wait failed, 12
<ERROR> 0 : Run 0000000F

This happens when Watchdog kills the program because kernel execution time exceeds a certain level. In that case, set the following environment variables.

export PZCLKernelTimeout=1000000000

The is specified in milliseconds.
